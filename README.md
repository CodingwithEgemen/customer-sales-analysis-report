# Sales Analysis Report

This repository contains a customer-based sales analysis project prepared using Google Sheets and exported to Excel.

The objective of the project is to analyze sales performance, enrich sales data with customer information, and generate insights using pivot tables and visualizations.

---

## Project Structure

- Sales data
- Customer master data
- Data enrichment using XLOOKUP
- Customer-based sales aggregation with pivot tables
- Sales visualization using column charts
- Results and insights documented in both Turkish and English

---

## Key KPIs

- Total Sales Amount
- Number of Customers
- Unmatched Records ("Not found")

---

## Tools Used

- Google Sheets
- Microsoft Excel
- XLOOKUP
- Pivot Tables
- Charts

---

## Notes

- Some sales records could not be matched with customer master data and are marked as **"Not found"**.
- Reports are provided in **both Turkish and English** versions.
